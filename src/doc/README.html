Example install on CentOS with yum and rpm

Download the rpms for story-xml, fop, and jdk.

yum install perl libxslt xdialog tidy perl-Config-General
rpm -ivh jdk-6u10-linux-i586.rpm
rpm -ivh fop-0.95-1-linux-2.6-i386.rpm
rpm -ivh story-xml-0.4.1-1-linux-2.6-i386.rpm
